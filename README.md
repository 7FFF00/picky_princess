# Picky Princess
Terminal based Roguelike dungeon crawler, themed around cooking and with a pastel based, cute aesthetic.

## Libraries
bearlibterminal
cereal (to be implemented)

## Build
On Linux:
```
g++ src/*.cpp -o princess -Iinclude -Llib -lBearLibTerminal -Wall
```
